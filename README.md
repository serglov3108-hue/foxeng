# FOXeng Platform

## Как запустить проект локально:
1. Установите Node.js
2. Выполните `npm install`
3. Запустите `npm run dev`
4. Откройте `http://localhost:3000`

## Как выложить в интернет (Бесплатно):
1. Создайте репозиторий на GitHub и загрузите туда эти файлы.
2. Зайдите на [Vercel](https://vercel.com).
3. Нажмите "Add New" -> "Project".
4. Импортируйте ваш репозиторий.
5. Нажмите "Deploy".

Сайт будет доступен по вашему уникальному адресу (например, `foxeng.vercel.app`).
